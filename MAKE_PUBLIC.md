# Make GitHub Repository Public

1. Go to: https://github.com/michettik/HomeAssistant-DSC-IT-100/settings

2. Scroll all the way down to the **Danger Zone** section

3. Click "Change visibility"

4. Select "Make public"

5. Type the repository name to confirm: HomeAssistant-DSC-IT-100

6. Click "I understand, change repository visibility"

Done! Now HACS will be able to access it.
